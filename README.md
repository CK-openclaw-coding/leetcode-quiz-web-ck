# LeetCode Quiz System Optimized

This project is an optimized version of a quiz system for LeetCode (or general technical) questions.

## Features
- **Auto-Loading**: Automatically loads `questions.json` and `answers.json` from the root directory.
- **Persistence**: Remembers your wrong questions and progress via `localStorage`.
- **Modern UI**: A clean, responsive design with dark mode support.
- **Searchable**: Easily find questions by keywords.
- **Offline Support**: Manual file upload fallback.

## Setup
1. Place your `questions.json` and `answers.json` in the root folder.
2. Open `index.html` in a browser or host via GitHub Pages.
